## ğŸš€ Getting Started

### Prerequisites
- A modern web browser
- Python (for local development server) or any HTTP server

### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/iprajwalsingh7/portfolio-website.git
   cd portfolio-website
   ```

2. **Run locally**
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Or using Node.js
   npx serve .
   
   # Or using PHP
   php -S localhost:8000
   ```

3. **Open in browser**
   ```
   http://localhost:8000
   ```
